# Plateforme BLOC ESTM

**Plateforme BLOC ESTM** est un espace collaboratif de communication interne dedie aux etudiants et au personnel de l ESTM. Elle facilite le partage d informations et la collaboration sur differents sujets academiques.

## Fonctionnalites Principales
- Fil d actualites collaboratif
- - Partage de ressources et de documents
  - - Espaces de discussion par themes
    - - Notifications internes
     
      - ## Stack Technique
      - - Backend : PHP
        - - Database : MySQL
          - - Frontend : CSS / JS
